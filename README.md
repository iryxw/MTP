## Описание

В рамках лабораторной работы реализована контейнеризация двух приложений на разных языках программирования:

- **Python-приложение** — REST API на базе Flask для управления задачами (to-do list) с поддержкой CRUD-операций.
- **Rust-приложение** — интерактивный трекер расходов, дополненный минимальным HTTP-сервером.

Оба сервиса упакованы в Docker-контейнеры с использованием многоэтапной сборки.  
Приложения запускаются одновременно через `docker-compose`.

## Используемые технологии и языки

- **Python 3.12** + Flask + Gunicorn
- **Rust** (edition 2021) — стандартная библиотека
- **Docker** + multi-stage build
- **docker-compose** (v3.9)
- Финальный образ Rust собран на базе `scratch`
